# HIT-CS-Courses

哈工大计算学部课程相关学习链接集合

> ### 说明
> 1. 欢迎`Pull Request`和`Issue`，推荐学长学姐们优秀的笔记博客和实验代码，或新增课程/专业；
> 2. 所有实验和笔记可以不完全是原创，但若借鉴较多请直接贴上原repo；
> 3. 学弟学妹们谨慎clone，毕竟如果大家重复率太高，老师改课程设计，对大家都不是好事哦；
> 4. 在GitHub搜索到仓库/CSDN搜索到博客后，进入作者主页会有更多学习资源；
>
> p.s. 此处鄙人先献丑，期望优秀的同学们前赴后继
